# 🧭 토론나침반 (Debate Compass)

고등학생들이 시사 이슈(뉴스)로부터 신뢰성 높은 토론 주제를 찾고, 발표 및 토론에 즉시 사용할 수 있는 찬반 근거 자료를 체계적으로 수집·검증해 주는 토론 도우미 웹 애플리케이션입니다.

---

## 🛠️ 기술 스택 및 아키텍처

- **Frontend**: React / Next.js (App Router, TypeScript)
- **Styling**: Vanilla CSS (CSS Variables 기반 프리미엄 다크/라이트 테마 제공)
- **Backend**: Next.js Route Handlers (서버 사이드 외부 API 호출)

---

## 🔒 보안 및 API 연동 규칙 (중요)

본 애플리케이션은 학생들의 학습 정보 정합성을 보호하고 보안 취약점을 예방하기 위해 아래의 설계 규칙을 철저히 따릅니다.

1. **API Key 보안**: `NAVER_CLIENT_SECRET`, `TAVILY_API_KEY`, `GEMINI_API_KEY`와 같은 외부 API 비밀키는 **절대 브라우저(프론트엔드)에 노출되지 않도록** 설계되었습니다. 모든 외부 호출은 Next.js 백엔드 API 라우트(`/api/...`) 내에서 안전하게 이루어집니다.
2. **환경변수 분리**: 실제 구동을 위한 API 키값들은 Git 버전 관리에 절대 업로드되지 않는 개인 환경변수 파일인 `.env` 혹은 `.env.local` 파일에 기재하여 보관합니다.
3. **Mock Fallback 시스템**: API 키가 아예 지정되어 있지 않거나 네트워크 통신이 실패하더라도, 시스템이 에러를 일으키지 않고 사전에 내장된 "AI 교육" 주제 기반의 고품질 **Mock 데이터로 매끄럽게 폴백(Fallback)되어 정상 구동**됩니다.

## ⚙️ 로컬 개발 환경 세팅 및 실행 방법

### 1. 환경 변수 파일 생성
1. 프로젝트 루트 디렉토리에 있는 `.env.example` 파일을 복사하여 `.env.local` 파일을 만듭니다.
2. **주의**: `.env.local` 파일은 로컬 검증 전용이며, 보안을 위해 절대 Git 리포지토리에 올리지(push) 마십시오.
3. 기호에 따라 다음 외부 API Key를 채워 넣습니다:
   ```env
   NAVER_CLIENT_ID=네이버_개발자_센터_아이디
   NAVER_CLIENT_SECRET=네이버_개발자_센터_시크릿
   TAVILY_API_KEY=Tavily_검색_API_키
   GEMINI_API_KEY=구글_AI_스튜디오_Gemini_키
   ```

### 2. 개발 서버 구동
프로젝트 루트 폴더(`토론나침반_00`)에서 터미널을 열고 아래 명령을 실행합니다.

```bash
# 개발 서버 실행
npm run dev
```

서버가 켜지면 웹 브라우저를 열고 **[http://localhost:3000](http://localhost:3000)** 으로 접속하십시오.

## 🛡️ 4단계 교차 정합성 검증 엔진 소개

학생들이 잘못된 AI의 답변(할루시네이션)에 속아 학교 토론회나 발표에서 틀린 정보를 활용하지 않도록, 시스템에 다음과 같은 4단계 정합성 검증 배지가 표기됩니다.

1. **뉴스 ↔ 토론 질문 정합성**: 추출된 추천 질문이 뉴스의 실제 사회적 갈등 흐름과 부합하는지 검증합니다.
2. **토론 질문 ↔ 자료전체 정합성**: Tavily에서 검색 수집된 웹 문서들이 논제 논쟁에 충분한 자료인지 평가합니다.
3. **자료 요약 ↔ AI 활용 포인트 정합성**: AI가 짠 교육 지침이 원문의 팩트를 왜곡하지 않고 결합되었는지 체크합니다.
4. **출처 신뢰도 ↔ 실제 출처 유형 정합성**: 도메인 파서 코드를 통해 공공기관/학술/언론 보도 신뢰도를 분류해 등급 배지를 노출합니다.
